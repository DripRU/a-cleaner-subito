# A cleaner Subito

A cleaner Subito è un'estensione per i browser Chromium-based che nasconde gli annunci promossi contrassegnati come “Vetrina” durante la navigazione tra i risultati di ricerca su Subito.it.

## Features

- Nasconde gli annunci promossi contrassegnati come “Vetrina”
- Consente di attivare o disattivare il filtro dal popup dell'estensione
- No tracking, richieste esterne o raccolte di dati

## Installazione

1. Scarica o clona questa repository
2. Apri il browser
3. Go to `NOME_DEL_BROWSER://extensions/` (es. chrome://extensions/)
4. Abilità la Modalità Sviluppatore
5. Clicca `Carica estensione non pacchettizzata`
6. Seleziona la cartella del progetto

## Compatibilità browser

L'estensione è fatta per i browser Chromium-based che usano Manifest V3.

Testata su:
- Google Chrome
- Brave

Dovrebbe funzionare perfettamente su tutti gli altri browser Chromium-based come **Microsoft Edge**, **Opera**, **Vivaldi**, etc.
